# CauNlpResume
## CAU Team Project - Resume redesign with NLP
* 팀장 : 임기찬
* 팀원 : 강민수 김상헌 오준오 천영재

## How to Run
* 우분투 환경: 하단에 기재해놓은 필요한 라이브러리들을 내려받고 '''python3 pyqtui.py'''를 실행한다.

## Enviornment
* OS : Ubuntu 18.04
* Language : Python 3.6
* Libraries : PANDAS, PyQt5, KoNLPy, Selenium, XlsxWriter
